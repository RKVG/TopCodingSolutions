Given the number of seconds since midnight, return a formatted String giving the number of hours, minutes, and seconds that have elapsed.

09: public class Time {
10: 
11:     public static final int SECONDS_PER_MINUTE = 60;
12:     public static final int SECONDS_PER_HOUR = SECONDS_PER_MINUTE * 60;
13: 
14:     public String whatTime(int seconds) {
15: 
16:         /*
17:         Get the whole number of hours that fits into seconds.
18:         Store that as h and then subtract the hours portion
19:          off of seconds.
20:          */
21:         int h = seconds / SECONDS_PER_HOUR;
22:         seconds -= (h * SECONDS_PER_HOUR);
23: 
24:         // Repeat to get the number of minutes.
25:         int m = seconds / SECONDS_PER_MINUTE;
26:         seconds -= (m * SECONDS_PER_MINUTE);
27: 
28:         // What's left over is the number of seconds.
29:         int s = seconds;
30: 
31:         // Create the return String in the appropriate format.
32:         return new String(h + ":" + m + ":" + s);
33:     }
34: }
Notes:
Remember that when dividing ints in Java, the / operator returns the whole number of times that the denominator fits into the numerator. So 10 / 3 = 3.

On line 21, we use this to get the number of hours in the input parameter. The result is multiplied by the number of seconds in an hour, and then subtracted from seconds (line 22).

At that point, the number of seconds left must be less than 1 hour. On line 25 and 26, we repeat the process to get the minute portion. Whatever is left over is the number of seconds.
