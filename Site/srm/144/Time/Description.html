Overview:
Given the number of seconds since midnight, return a formatted String giving the number of hours, minutes, and seconds that have elapsed.
Java Source:
    
    01: public class Time {
    02:
    03:     public static final int SECONDS_PER_MINUTE = 60;
    04:
    05:     public static final int SECONDS_PER_HOUR = SECONDS_PER_MINUTE * 60;
    06:
    07:     public String whatTime(int seconds) {
    08:
    09:         /*
    10:         Get the whole number of hours that fits into seconds.
    11:         Store that as h and then subtract the hours portion
    12:          off of seconds.
    13:          */
    14:         int h = seconds / SECONDS_PER_HOUR;
    15:         seconds -= (h * SECONDS_PER_HOUR);
    16:
    17:         // Repeat to get the number of minutes.
    18:         int m = seconds / SECONDS_PER_MINUTE;
    19:         seconds -= (m * SECONDS_PER_MINUTE);
    20:
    21:         // What's left over is the number of seconds.
    22:         int s = seconds;
    23:
    24:         // Create the return String in the appropriate format.
    25:         return h + ":" + m + ":" + s;
    26:     }
    27: }
Notes:

Remember that when dividing ints in Java, the / operator returns the whole number of times that the denominator fits into the numerator. So 1 0 / 3 = 3.

On line 14, we use this to get the number of hours in the input parameter. The result is multiplied by the number of seconds in an hour, and then subtracted from seconds (line 15).

At that point, the number of seconds left must be less than 1 hour. On line 18 and 19, we repeat the process to get the minute portion. Whatever is left over is the number of seconds.
